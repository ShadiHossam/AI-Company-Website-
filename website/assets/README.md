# Website Assets Guide

This directory holds the graphic assets for the website. To prevent 404 resource errors and ensure proper branding/social sharing, please place your custom images here with the following filenames:

## Required Brand Assets

- `logo.png`: The company logo displayed in the navbar, footer, and emails.
- `favicon.ico` / `favicon.png`: The website favicon displayed in browser tabs.

## Required Open Graph (OG) Images

These are the social share cards shown when links are shared on LinkedIn, WhatsApp, Twitter, etc. The recommended size is **1200 x 630 pixels**:

- `og-home.jpg`: Share card for the Homepage.
- `og-about.jpg`: Share card for the About page.
- `og-blog.jpg`: Share card for the Blog page.
- `og-contact.jpg`: Share card for the Contact/Consultation page.
- `og-products.jpg`: Share card for the Products page.
- `og-pricing.jpg`: Share card for the Pricing page.
- `og-results.jpg`: Share card for the Case Studies/Results page.
- `og-services.jpg`: Share card for the Services overview page.
